

## First EVN


### 1、install cargo

```shell
curl https://sh.rustup.rs -sSf | sh
```

### 2、install cargo-tauri

```shell
cargo install tauri-cli --git https://github.com/tauri-apps/tauri
```


### 3、run vue

```shell
yarn run serve   # default port is 8080 at tauri.conf.json
```

### 4、run app dev

```shell
cargo-tauri dev
```
